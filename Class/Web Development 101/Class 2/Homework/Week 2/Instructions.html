JavaScript Homework 2

For this assignment, you will complete some JS tasks to get you accustomed to variable scopes and some JQuery tasks.

1. Simple Output

Part 1: Notice there are two files provided for part1:
  • html file called part1.html
  • javascript file called part1scripts.js
These files have the first bit of code to get started.
Part 2: Create 3 new functions in part1scripts.js:
  • harvestApples()
  • harvestOranges()
  • harvestBananas()
Part 3: Make harvestApples() simply add the following string to the inner HTML of the body in part1.html: numberOfFruit + " apples harvested! </br>"
Part 4: Make harvestOranges() declare its own numberOfFruit variable set equal to 3, with var numberOfFruit = 3;
Part 5: After declaring the new numberOfFruit variable make harvestOranges() add the following string to the inner HTML of the body in part1.html: numberOfFruit + " oranges harvested! </br>"
Part 6: Make harvestBananas() modify the global numberOfFruit variable equal to 0, with numberOfFruit = 3;
Part 7: After modifying the numberOfFruit variable make harvestBananas() add the following string to the inner HTML of the body in part1.html: numberOfFruit + " bananas harvested! </br>"
Part 8: call your functions in this order in part1scripts.js after the functions have been created in the code:
  1. harvestApples()
  2. harvestOranges()
  3. harvestBananas()
  4. harvestApples()
  5. harvestOranges()
  6. harvestBananas()
Part 9: Notice that you called your functions each twice and in the same pattern: apples, oranges, bananas, apples, oranges, and bananas. Add a paragraph element to the body in part1.html explain why the second call on harvestApples has a different number than the first call, even though harvestApples never modifies the numberOfFruit. See scopes: https://www.w3schools.com/js/js_scope.asp
