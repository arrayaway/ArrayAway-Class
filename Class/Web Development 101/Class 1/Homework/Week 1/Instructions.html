CSS Homework

Part 1: Make it where all b elements have the color blue by adding that style in main.css.
Note: Main.html is already referencing main.css, so you should be able to add b elements in that file and see them blue.
Note: Add a b element to Main.html if it does not already have one to check that your css works.
Part 2: Create the following three classes in main.css and add them to a single div element in Main.html:
  • bg-red
    • Sets the background to red.
  • big-block
    • Sets the display to block.
    • Sets the width to 500px.
    • Sets the height to 500px.
  • padded
    • Sets the padding to the left to 20px.
    • Sets the padding to the right to 20px.
    • Sets the padding to the top to 50px.
Part 3: Make it where the first letter of a paragraph p element that is the child of a div element is bold and red by adding that style in main.css.
Note: Add a div element with a p element in it with text to Main.html if it does not already have one to check that your css works.
