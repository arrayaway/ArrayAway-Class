Login Page Homework

For this assignment, you will make the login page for your social media website! The objective is to make a Login.html page that looks as close to the login below as possible. There are steps below the image to get you started.
Part 1: In a folder you can find later, create two files:
  • html file called login.html
  • css file called main.css
Part 2: Add the base html code to login.html. (IE the doctype, the html element, the head, and the body)
Part 3: Include your main.css reference and bootstrap by adding the bootstrap references inside the head element (refer to previous homework assignments).
Note: the background color of the login page in the image appears to be a light blue.
Part 3: In main.css set the body's background to some sort of light blue. Perhaps #e9faff?
Part 4: Create the white box containing the login form class:
  1. Create a new css class in the main.css file to add the container styles to. Name it "login-container".
  2. Make the class have a height of 400px.
  3. Make the class have a background color of white.
  4. Make the edges of the container class rounded by modifying the border-radius property.
  5. Looks like all of the text is centered, so make the class center its text with text-align: center.
  6. Add a shadow effect using the box-shadow property.
Part 5: In the body of login.html, add a div with the class "container". Then inside this div add a div with the class of "row". Then inside this div, add another div with 2 classes: col-md-6 and the class we just created: "login-container".
Note: if you look at your login.html page in a browser, you should see that your white container is off to the left rather than in the center. Let's fix that.
Part 6: Add another class, "offset-md-3", to the div element that already has the two classes "col-md-6 login-container". Add the class before the other two inside the quotes.
Note: if you look at your login.html page in a browser, you should see that your white container is also very close to the top of the page.
Part 6: Modify the login-container class in main.css so that it sets the margin-top to 40px.
Part 7: Inside the login-container div in login.html, add an h2 element with the text "Account Login".
Part 8: After the h2 element, add a form element that contains 2 inputs and a button:
  1. An input of type text.
  2. An input of type password.
  3. A button of type submit.
Refer to https://getbootstrap.com/docs/4.0/components/forms/? to make the inputs bootstrapped styled.
Refer to https://getbootstrap.com/docs/4.0/components/buttons/ to make the button bootstrapped styled. Look into large buttons and block level buttons.
Part 9: finally after the form, use p, a and br tags to add the last bit of content as seen in the image.
Note: if you look at your login.html page in a browser, you should see that the content in your white container is really close to the edges. Let's fix that.
Part 10: In main.css, modify the login-container class so that it has a padding of 50px.
Bonus: Now that you have a good base login page, let your creative juices flow and style the page as your own. Change the background-color, make the background an image, add more, make it darker, do whatever you want to make the page more your style!
